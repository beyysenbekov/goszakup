# 🤖 Бот мониторинга Госзакупок — Область Абай / Район Мақаншы

Работает **БЕЗ API токена Госзакупок** через публичный REST API.

---

## 🚀 Деплой на Railway

### Шаг 1 — Получите токен Telegram-бота
1. Напишите [@BotFather](https://t.me/BotFather) → `/newbot`
2. Введите имя и username бота
3. Скопируйте токен вида `1234567890:AAHxxx...`

### Шаг 2 — Узнайте свой chat_id
1. Напишите боту [@userinfobot](https://t.me/userinfobot)
2. Он ответит вашим числовым ID

### Шаг 3 — Загрузите на GitHub
Создайте репозиторий и загрузите все файлы.

### Шаг 4 — Деплой на Railway
1. Зарегистрируйтесь на [railway.app](https://railway.app)
2. **New Project → Deploy from GitHub repo**
3. Выберите ваш репозиторий
4. Перейдите в **Variables** и добавьте:

| Переменная | Значение |
|---|---|
| `BOT_TOKEN` | токен от @BotFather |
| `ADMIN_CHAT_ID` | ваш числовой chat_id |

5. Railway сам установит зависимости и запустит бота.

---

## ⚙️ Как работает фильтр без токена

Бот использует **публичный REST API** `ows.goszakup.gov.kz/trd-buy/all` (без авторизации).
Для каждого объявления запрашиваются лоты и проверяется **КАТО-код** места доставки.
Все коды Абайской области начинаются с `63` — это надёжный фильтр.
Если API недоступен — автоматически включается резервный HTML-парсинг.
